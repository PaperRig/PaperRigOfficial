Professional Minecraft mod with smart optimizations. Download the API first, then install the mod versions.

Works on 1.21.x

System Requirements

  • Minecraft Version: 1.21.1 (stable)
  • Java: 17+
  • Platform: Paper or Fabric
  • API: Required for the mod to work

How to Install

  1. Download the API
  2. Download the mod version (v1 or v2)
  3. Place both files in your mods folder
  4. Launch Minecraft

Frequently Asked Questions

Do I need the API?

Yes — the mod will not work without the API.

Which versions are supported?

Stable on 1.21.1. Other versions are currently in testing.

Can I use both versions?

Yes, but only one version should be active at a time.

Is this safe?

Yes — always download from the official site and scan files if needed.

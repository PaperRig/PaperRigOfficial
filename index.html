System Requirements & Installation

System Requirements

  • Minecraft 1.21.1 (stable)
  • Java 17+
  • Paper or Fabric
  • API required

How to Install

  1. Download the API
  2. Download the mod version (v1 or v2)
  3. Place both files in your mods folder
  4. Launch Minecraft

Stable on 1.21.1. Other versions are currently in testing.

Can I use both versions?

Yes, but only one version should be active at a time.

Is this safe?

Yes — always download from the official site and scan files if needed.
